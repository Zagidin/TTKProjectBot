# Бот ТТК
___
### Запуск **Бота**
___
#### Windows

```shell
 python main.py
```
#### Linux

```shell
 python3 main.py
```

---
### Установка нужных библиотек

---

```shell
 pip install aiogram==2.23.1
```
```shell
 pip install python-dotenv
```
```shell
 pip install pyaudio
```
```shell
 pip install pydub
```

---

### Перед запуском, создайте файл `.env`
> API_TOKEN_BOT=<Ваш Токен Бота>

> DATABASE_URL=<Подключение к БД (`ПРИМЕР`: sqlite:///./test.db)>
